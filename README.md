# Научный калькулятор (C# / Windows Forms)

Полнофункциональное приложение-калькулятор с расширенными математическими возможностями, разработанное на **C#** с использованием **Windows Forms** (.NET Framework).

## Возможности

### Арифметика
- Базовые операции: +, -, ×, ÷, %
- Корректный учёт приоритета операций (`2 + 2 × 2 = 6`)
- Обработка цепочек вычислений без промежуточного нажатия =

### Научные функции
| Категория       | Функции                                  |
|-----------------|------------------------------------------|
| Степени/корни   | `x²`, `x³`, `xʸ`, `ʸ√x`, `√x`, `³√x`     |
| Логарифмы       | `ln`, `log`, `10ˣ`, `eˣ`                 |
| Тригонометрия   | `sin`, `cos`, `tan` (аргумент в радианах)|
| Прочее          | `x!`, `1/x`, `±`, `%`                    |

### Управление памятью
- `MS` — сохранить значение в память
- `MR` — вставить значение из памяти
- `M+` / `M-` — добавить/вычесть текущее значение к памяти
- `MC` — очистить память
- `M` — показать текущее значение памяти

### Удобство использования
- `←` — удалить последний символ
- `С` — очистить текущий ввод
- `CE` — полный сброс (очистка стеков операций)
- Автоматическая защита от дублирования разделителя (`,`)
- Понятные сообщения об ошибках

## Техническая реализация
Реализован **стековый вычислитель** с динамическим учётом приоритета операций:
-private List<double> numbers = new List<double>();    // операнды
-private List<string> operations = new List<string>(); // операторы
